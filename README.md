# ubiquitiMfiPower
Used to connect and control a Ubiquiti device  (Ubiquiti mFi Power Pro 6 slots)

# What I Used?
* Node.js
* Express.js
* ssh2.sj
* Unix commands
* HTML

Run Command:

	npm install
  
Then, edit the index.js file, and write the ip of your device in [[DEVICE_IP]] spot. After that just run:

	npm start index.js
